
Desfazendo burradas localmente(antes de enviar pra nuvem):

-----------git reset

comando: git reset NomeDoArquivo


serve pra reverter o envio pro staging area
em outras palavras, eu modifiquei um arquivo, enviei pra staging area com o comando: git add arquivo,
porém me arrependi e quero tirá-lo da staging area, uso o comando git reset arquivo.




-----------git checkout 
e se eu quiser reverter/apagar a última modificação de um determinado arquivo que ainda não foi pra staging área, 
ou que até foi, mas eu tirei de lá com o git reset ?
uso o comando: git checkout aula

-------git revert

funciona depois de enviar pra nuvem)
